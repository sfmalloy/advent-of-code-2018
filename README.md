# Advent of Code 2018!!!

These are my solutions to Advent of Code 2018.
If you want to download for yourself, compile using make and the run.

**Running**
----------------------------------
For single day:
    
    $ ./Advent -s <day_number>
    
For all days:
    
    $ ./Advent -a
